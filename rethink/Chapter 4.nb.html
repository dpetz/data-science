Chapter 4

Linear model of height

Update R version (currently 3.6.2.and install book packages:

install.packages(c("coda","mvtnorm","devtools"))
library(devtools)
devtools::install_github("rmcelreath/rethinking")

Load !KungSan data set created by Nancy Howell in the 1960s:

library(rethinking)
data(Howell1)
d <- Howell1
str(d)
'data.frame':   544 obs. of  4 variables:
 $ height: num  152 140 137 157 145 ...
 $ weight: num  47.8 36.5 31.9 53 41.3 ...
 $ age   : num  63 63 65 41 51 35 32 27 19 54 ...
 $ male  : int  1 0 0 1 0 1 0 1 0 1 ...

Model: \[ h_i \sim Normal(\mu,\sigma) \\ \mu \sim Normal(178,20) \\ \sigma \sim Uniform(0,50) \] Prior for height’s mean:

curve(dnorm(x,178,20),from=100,to=250)

Prior for height’s variance:

curve(dunif(x,0,50),from=-10,to=60)

Simulate height by sampling from prior:

sample_mu <- rnorm(1e4,178,20)
sample_sigma <- runif(1e4,0,50)
prior_h <- rnorm(1e4,sample_mu,sample_sigma)
dens(prior_h)
